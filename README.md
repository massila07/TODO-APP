# Todo List App (JavaScript)

A simple Todo List application built with **JavaScript, HTML, and CSS**, demonstrating frontend-backend interaction and basic CRUD operations.

---

## Features

- Add new tasks to your todo list
- Display all tasks dynamically
- Delete tasks
- Data is stored temporarily in a backend server (Node.js + Express)
- Interactive and responsive interface

---

## Technologies

- JavaScript (ES6)
- Node.js + Express for backend
- HTML / CSS for frontend
- Fetch API for client-server communication
